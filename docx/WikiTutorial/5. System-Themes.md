# 🎨 Themes and Visuals

We provide three distinct visual experiences for our users:

1. **Regular Light Theme**: A clean, professional look for everyday browsing.
3. **Dark Theme**: A sleek, high-contrast look using advanced CSS transitions and box shadows.
2. **Sport Theme**: A brighter colourful contrast of sporty red and blue.

How to switch? The site uses external CSS files that swap dynamically based on your selection or the current season, ensuring the site always feels fresh.
