# 🎨 Themes and Visuals

We provide three distinct visual experiences for our users:

1. **Regular Theme**: A clean, professional look for everyday browsing.
2. **Thanksgiving/Holiday Theme**: Warm tones and festive decorations.
3. **Dark/Modern Theme**: A sleek, high-contrast look using advanced CSS transitions and box shadows.

How to switch? The site uses external CSS files that swap dynamically based on your selection or the current season, ensuring the site always feels fresh.
