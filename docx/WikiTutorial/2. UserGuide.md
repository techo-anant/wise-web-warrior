# 📖 User Guide: How to Navigate

Getting started with CarDeals is simple. Here is how you can use the platform:

1. **Browsing the Catalogue**: Use the main menu to access the "Inventory" page. You can scroll through our 20+ vehicle listings.
2. **Customizing Options**: Click on any vehicle to see specific options (e.g., changing the Engine Type or Trim level).
3. **Responsive Design**: Our site works on desktops, tablets, and phones. The menu will automatically collapse into a "hamburger" icon on smaller screens.
4. **Theme Switching**: You can change the website's look (e.g., Seasonal Themes) via the settings toggle in the navigation bar.
