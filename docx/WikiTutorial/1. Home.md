# 🚗 About CarDeals Catalogue

Welcome to **CarDeals** — a full-stack dynamic web platform built for automotive enthusiasts, buyers, and dealership administrators.

This project is a comprehensive digital vehicle catalogue backed by a **React** frontend and a **Node.js + Express** backend connected to a **MySQL** database. Users can browse, search, filter, compare, and save vehicles from a live inventory of 20+ car listings. Admins have a full management interface for cars, users, and site settings.

---

## 🧱 What the Platform Includes

### For Users
- Browse **20+ vehicle listings** with photos, specs, pricing, and video tours
- Filter inventory by price, mileage, category (SUV, Sedan, Truck, Electric, Coupe, Van), year, and condition
- View detailed car pages with a scrollable image gallery, HTML5 video player, and full specification table
- Compare vehicles side-by-side on a dedicated comparison page
- Use the loan/financing calculator to estimate monthly payments
- Save favourite cars to a personal wishlist
- Receive price drop and inventory match notifications
- Find dealership locations on an interactive map (Leaflet.js)

### For Admins
- Manage the full car inventory — add, edit, and delete listings
- Manage user accounts — view, disable, or promote users to admin
- Switch and preview the site's active visual theme
- Monitor live health status of the database and API services

## 🗄️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React (Create React App), React Router, Axios, Leaflet.js |
| Backend | Node.js, Express.js |
| Database | MySQL (via mysql2 connection pool) |
| Auth | JWT (JSON Web Tokens), bcrypt password hashing |
| Styling | CSS Modules + 3 swappable external theme files |
| Charts | Custom chart components (PriceHistoryChart, InventoryChart) |

## 🗃️ Database

Three core tables power the platform:

- **`users`** — Stores accounts with name, email, hashed password, role (`user` / `admin`), and active status
- **`cars`** — Stores all vehicle listings with make, model, year, price, mileage, category, tag, fuel type, engine, images, and availability
- **`dealers`** — Stores dealership branch locations with address, GPS coordinates, phone, email, and hours
